APANTIAS

A Python ANalysis Tool for In-depth sensor Array Statistics

Apantias GitHub Repository

The Apantias software package performs raw data analysis of a frame-based pixel detector, which provides signals for each of 64×64 pixels (in analogue digital units, ADU) in discrete time steps. Each time step corresponds to one frame and reflects the collection and storage of signal charges within each pixel of the DEPFET detector.

In these detectors, the signal is cleared during the readout process, which is performed in a rolling shutter mode (with one row of 64 pixels read out in parallel). In the case of RNDR-DEPFETs, each signal is read out multiple times to reduce noise by averaging the readings. The Apantias software supports analysis with both single and multiple repetitions.
